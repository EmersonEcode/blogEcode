Sobre o Projeto Ecode

Uma jornada de aprendizado em desenvolvimento Full-Stack.

Bem-vindo ao Ecode! Este blog não é apenas uma coleção de artigos de tecnologia; é a documentação viva da minha jornada de aprendizado e aprimoramento em programação.

O principal objetivo deste projeto é servir como um campo de treinamento prático. Cada funcionalidade que você vê, do sistema de posts ao design responsivo, foi construída do zero como um exercício para aprofundar meus conhecimentos em tecnologias modernas.

Tecnologias Utilizadas

Este blog foi construído com:

  • Frontend: Angular v17+ (Standalone Components, RxJS, SCSS)
  • Backend (Em breve): Node.js com Express/NestJS (planejado)
  • Banco de Dados (Em breve): PostgreSQL ou MongoDB (planejado)
  • Deploy: Vercel/Netlify
